Anatomy Reference

This section offers an interactive reference to each anatomical term identified in the Zebrafish Atlas. To view an anatomical part within the context of a virtual slide, choose a term from the list below. A virtual slide, with capabilities to zoom and scroll, illustrates the term. The definition and function of each term will be given. Where applicable, a link to the term at the ZFIN website is also included.

Select a term from the list below to begin.
